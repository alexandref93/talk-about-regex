---

match

"say hello world".match(/hello\s(world)/) => ["hello world", "world"]

---

valid

/hello\sworld/.test("say hello world") => true

/hello\sworld/.test("say hello fulano") => false

---

replace

"say hello world".replace(/world/, "fulano") => "say hello fulano"
